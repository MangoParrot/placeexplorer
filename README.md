# Place Explorer
This is a github repository made to explore r/place

A little late to the bandwagon, I know.

Written in python
Requirements:
 - Python 3.xx
 - PyAutoGui
 - Pygame
 - Pandas
 - sys
 - numpy

# Installation
Download as a zip, pip install the required libraries and run the Main.py function
